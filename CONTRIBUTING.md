# Contributing to R6 Accessory Corrector

Thank you for wanting to contribute!

## Basic Rules

* Do not break compatibility with R6
* Preserve attachments and accessory positions
* Always test with multiple accessories before submitting a PR

## Code

* Use clean and readable Luau
* Avoid hardcoding values
* Always handle errors (pcall)

## Commits

* Use clear messages:

  * fix: fixes MeshPart scaling
  * feat: adds support for batch conversion

## Pull Requests

* Explain what has been changed
* Show before/after if possible (images help a lot)
* Do not submit untested code

## Contribution Ideas

* Improve scale calculation
* Support for more accessory types
* Performance optimization
* Automatic system for all players

Thanks for contributing 🚀
